Installing Git on Windows

1. git website: https://git-scm.com/
- download Git for Windows 64-bit (2.10.0)
- free book: Pro Git (https://git-scm.com/book/en/v2)
- release notes: C:\Program Files\Git\ReleaseNotes.html

2. Set up configurations
-  file c:\ProgramData\Git\config with WordPad

- Change user name, email, text editor"
git config --global user.name "Kevin Harris"
git config --global user.email kcshadow@comcast.net
(64-bit) git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -nosession"
(32-bit) git config --global core.editor "'C:/Program Files (x86)/Notepad++/notepad++.exe' -multiInst

- Check configuration settings
git config --list


3. Set up repository
- create directory, change directory, initialize as Git repository (git init)
  can use "dir /AH" to see file .git

4. Add files to repository and initial commit.
- move files to directory
- check status of repository (git status)
- add each file to staging (git add filename, or git add *.c, etc).
- check status again to be sure files were added (git status)
- perform initial commit (git commit -m "Initial Commit") - must use double quotes on windows.

5. Check in changes 
- check status to see if files have changed (git status)
- use (git add filename) to begin tracking new files
- Also (git add filename) to stage modified files
  Note: the modified file appears in green when staged (read for check in).
- commit changes (git commit)
- check git log for change messages (git log)

6. Remove files
- (git rm)
